# MarktMenu Live - Safe Demo

Veilige algemene demo voor marktkramen, foodtrucks en cateraars.

Deze versie:
- gebruikt geen echte bedrijfsnaam;
- gebruikt geen echt logo;
- gebruikt geen echte contactgegevens;
- gebruikt geen echte standplaatsen;
- draait in DEMO_MODE;
- haalt geen Supabase-data op;
- gebruikt lokale demo-data in de browser;
- admin login: pincode `2468`.

Controle na deploy:
- `/app.js` begint met `/* safe-demo-v1 */`
- `/styles.css` bevat `safe-demo-blue-v1`
- `/config.js` heeft `DEMO_MODE: true`

Gebruik deze versie voor koude acquisitie of als algemene pitchdemo.
